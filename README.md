# Purpose
BLEListerSwift is an iOS application that lists Bluetooth Low Energy devices.

# App Requirements

## iOS
An iOS device with Bluetooth Low Energy (BLE). E.g. iPhone >= 4s.
iOS 10.3 or newer.
